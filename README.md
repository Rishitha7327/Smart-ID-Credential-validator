Student Validation System (Python)

This Python program validates Student ID, Email ID, Password, and Referral Code.
If all inputs match the required format, it prints Approved, else Rejected.

Validations

Student ID: CSE-123 (7 chars, starts with CSE-, last 3 digits)

Email: must contain @ and ., and end with .edu

Password: min 8 chars, starts with uppercase, contains at least 1 digit

Referral Code: REF12@ (6 chars, starts with REF, 2 digits, ends with @)
